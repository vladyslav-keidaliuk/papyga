# Papyga
Telegram bot that reminds you of upcoming regular events

Даний бот названо «Папуга», бо основна його мета – нагадувати учням/студентам/працівникам про певні регулярні події шляхом надсилання повідомлення у групу за х хвилин/годин до події (це можна налаштовувати під себе). Згодом же з’явились додаткові розважальні функції, які можна прибрати, або перефразувати, як Вам буде потрібно.

На даний момент бот має 6 команд:
-	tleft|
У відповідь на дану команду бот відправить у чат інформацію про те, скільки у % відношенні (0-100) заняття/пари/уроку/регулярної події вже минуло.

-	upgrade|
У відповідь на дану команду бот відправить повідомлення:
1) з інформацією про режим та надасть користувачу випадковий розмір в межах, які зазначаються (при першому виклику команди)
2) про те, що розмір змінився (якщо команда викликається не вперше користувачем та таймер вже минув)
3) про те, що користувач рано зайшов, та після якої дати та часу зможе знов спробувати (якщо команда викликається не вперше та таймер ще не минув)

-	whoi|
У відповідь на цю команду бот відправить повідомлення:
1) з інформацією про режим та надасть користувачу випадкову цитату та тотемну тварину(при першому виклику команди)
2) з новою цитатою та тотемною твариною на наступні 12 годин (якщо команда викликається не вперше користувачем та таймер вже минув)
3) про те, що користувач рано зайшов, та після якої дати та часу зможе знов спробувати (якщо команда викликається не вперше та таймер ще не минув)
-	top|
У відповідь на цю команду бот відправить повідомлення з топ 5 користувачів режиму «Байрактар».

-	iwannadie|
У відповідь на цю команду бот відправить повідомлення:
1) з інформацією про режим та надасть користувачу випадкову відстань в межах, які зазначаються (при першому виклику команди)
2) про те, що розмір змінився (якщо команда викликається не вперше користувачем та таймер вже минув)
3) про те, що користувач рано зайшов, та після якої дати та часу зможе знов спробувати (якщо команда викликається не вперше та таймер ще не минув)
-	topm|
У відповідь на цю команду бот відправить повідомлення з топ 5 користувачів режиму «Скелелази».
-------------------------------------------------------------
This bot is called "Papyga"(parrot) because its main purpose is to remind pupils/students/employees about certain regular events by sending a message to the group x minutes /hours before the event (this can be customized). Later, additional entertainment functions appeared, which can be removed or rephrased as you need. 

At the moment, the bot has 6 teams:
-	tleft| 
In response to this command, the bot will send information to the chat about how much in % (0-100) the class/pair/lesson/regular event has already passed.

-	upgrade|
In response to this command, the bot will send a message:
1) with information about the mode and will provide the user with a random size within the limits that are specified (at the first command call)
2) that the size has changed (if the command is not called for the first time by the user and the timer has already expired)
3) about the fact that the user logged in early, and after what date and time he can try again (if the command is not called for the first time and the timer has not yet expired)

-	whoi|
In response to this command, the bot will send a message:
1) with information about the mode and will give the user a random quote and a totem animal (at the first command call)
2) with a new quote and a totem animal for the next 12 hours (if the command is not called for the first time by the user and the timer has already expired)
3) about the fact that the user logged in early, and after what date and time he can try again (if the command is not called for the first time and the timer has not yet expired)
-	top
In response to this command, the bot will send a message from the top 5 users of the Bayraktar mode .

-	iwannadie|
In response to this command, the bot will send a message:
1) with information about the mode and will provide the user with a random distance within the limits that are specified (at the first call of the command)
2) that the size has changed (if the command is not called for the first time by the user and the timer has already expired)
3) about the fact that the user logged in early, and after what date and time he can try again (if the command is not called for the first time and the timer has not yet expired)
-	topm|
In response to this command, the bot will send a message from the top 5 users of the " Climbers " mode.
